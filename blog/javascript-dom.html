The differences between HTML and CSS

Two important elements to understand are HTML and CSS. HTML is the cake then CSS is the icing. An analogy is a great way for non-technical to establish a basic understanding of a complicated concept. An analogy that could be used to describe them both is the idea of a house. A house has rooms, doors, a roof and a floor. Similarly a website uses a type of room or sections for example a header, body and footer. HTML is what organises these areas into a hierarchy.



To make the rooms and everything inside of it look cool, unique and creative is where CSS comes into play. CSS describes colours, shapes and positions of certain elements in a website.



The way in which you interact with aspects of a home such as turning lights on or off or adjusting the temperature on the central heating is known as Javascript. Javascript coordinates the different elements between HTML and CSS.

The Dom

The Dom stands for Document Object Model It is a way that we can change HTML with Javascript. The DOM and HTML are like the same document but in two different languages. The DOM is everything attached to a document and also the ways in which we can interact with it. These changes can be made using methods, sub objects and properties.

Objects


Objects and arrays are known to be a representation of special data that is Mutable; it is used to store a collection of data, not just a single value. Arrays are a special type of variable that is also mutable and can also store lists of values. Objects are used to represent ‘things’. That thing could be a person, a vehicle or a building or a book. It is anything that can be made up or that can be defined by a set of characteristics called properties. Properties in objects can be accessed, added, changed, and removed by using either dot or bracket notation.

Arrays


Arrays are often used to store a list of different items in a single variable. They are very useful when they are used to create ordered collections. To access an item you can access it by whatever its numerical position is on the list. Arrays use what is called a zero-based indexing system, the first item of the array has an index of zero, the second has 1 etc.

Functions

In Javascript a function is similar to a procedure. It is a set of instructions that helps to perform tasks and calculate values. To be a function it takes an input and then gives a certain output. Javascript functions let you give a value to a block of code in which you can execute that particular function as many times as you want to.
